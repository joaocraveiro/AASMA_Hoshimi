Configuring MS Visual Studio

NOTE

In this text the following abbreviations are used:

  • {PROJECT_HOSHIMI_HOME} – The directory where you installed (unziped) Project Hoshimi SDK

A MS Visual Studio 2005 solution is provided with the AASMA Framework to edit the source code. Just open ProjectHoshimiAASMA.sln

Whenever you build the code a DLL is generated: AASMAHoshimi.dll. You must copy this DLL to the directory: {PROJECT_HOSHIMI_HOME}\players.

Optionally, you may configure MS Visual Studio to copy automatically as a "Post-build Event". In order to do this, follow these steps:

  1. In the Solution Explorer window, right-click on the AASMAHoshimi project (the one in bold) and choose Properties
  2. Select Build Events on the left section

  3. Edit the Post-Build Event command line, on the right section, to:

    copy "$(TargetPath)" "{PROJECT_HOSHIMI_HOME}\players" /y
