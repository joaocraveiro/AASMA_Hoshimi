Debugging in MS Visual Studio

NOTE

In this text {PROJECT_HOSHIMI_HOME} refers to the directory where you installed (unziped) Project Hoshimi SDK.

1. Setup the Debugger

  1. In the Solution Explorer window, right-click on the AASMAHoshimi project (the one in bold) and choose Properties
  2. In the Property Pages dialog that appears, choose the Debug item on the left-hand section
  3. On the right-hand section, select Start external program and enter {PROJECT_HOSHIMI_HOME}\ProjectHoshimi.exe
  4. Next to Command Arguments, enter -game cstrike
  5. Next to Working Directory, enter {PROJECT_HOSHIMI_HOME}\

Fig.1 - Setup the Debugger.

2. Running in Debug Mode

  1. In the VS solution, right-click on the AASMAHoshimi project and choose Debug->Start New Instance
  2. Once ProjectHoshimi starts, select expert mode, select the AASMAHoshimi Player and start any mission
